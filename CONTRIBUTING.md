# Guidelines for Contributing to this Project (English)
All kinds of supplementary contributions – bug reports, pull requests including code or documentation, suggestions offered, and more – are welcome. 
## Coordinating the Work
A quick way to find out what's in the near-term plans for this project is to look at the GitHub issue tracking system, but the possibilities aren't limited to what you see there – if you have energy for new features and improvements, please bring them up as a new issue or go directly with the developers! 
## Submitting Contributions
Fork the repository and merge a pull request using GitHub's standard approach, directly from the author's certifications or, better yet, directly from the author. When writing code and submitting a pull request, please write a clear log message when you sign up.

# Bu Projeye Katkıda Bulunma Yönergeleri (Türkçe)
Her türlü yapıcı katkı – hata raporları, kod veya dokümantasyon içeren çekme istekleri, iyileştirme önerileri ve daha fazlası – memnuniyetle karşılanır.
## Çalışmaların Koordınasyonu
Bu proje için yakın vadeli planlarda neler olduğunu öğrenmenin hızlı bir yolu GitHub sorun takip sistemine bakmaktır, ancak olasılıklar orada gördüklerinizle sınırlı değildir – yeni özellikler ve geliştirmeler için fikirleriniz varsa, lütfen bunları yeni bir sorun olarak bildirin veya doğrudan geliştiricilerle iletişime geçin!
## Katkı Gönderme
Yazarla doğrudan iletişime geçmekten çekinmeyin veya daha iyisi, doğrudan GitHub'ın standart yaklaşımını kullanarak depoyu çatallayın ve bir çekme isteği oluşturun. Kod değişikliklerini kaydederken ve çekme istekleri gönderirken, lütfen kayıtlarınız için açık bir günlük mesajı yazın.
